# FinSight

FinSight is an ongoing AI-powered financial statement auditing platform built to analyze uploaded statements and provide audit insights through a dashboard.

## Current Features
- User authentication (Register/Login with JWT)
- Dashboard frontend
- Financial statement upload
- Backend file storage
- Audit API integration
- JSON-based audit response display

## Tech Stack
### Backend
- Python
- Django
- Django REST Framework
- SQLite

### Frontend

- Bootstrap
