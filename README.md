# FinSight

FinSight is an ongoing AI-powered financial statement auditing platform built to analyze uploaded financial statements and generate audit insights through an interactive dashboard. The platform combines conventional audit workflows with LLM-powered analysis to help users extract meaningful insights from financial data.

## Current Features

* User authentication (Register/Login with JWT)
* Dashboard frontend
* Financial statement upload
* Backend file storage
* Audit API integration
* Frontend-backend integration
* LLM-based audit insight generation
* JSON-based audit response display
